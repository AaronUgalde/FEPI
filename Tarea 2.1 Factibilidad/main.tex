\section{Introducción}

El estudio de factibilidad se define como el análisis que permite determinar si un proyecto es viable desde múltiples perspectivas antes de comprometer recursos significativos \cite{baca2013}. Según Sapag Chain, este proceso busca reducir el riesgo asociado a la inversión mediante la recopilación de información relevante sobre el entorno y las capacidades internas \cite{sapag2011}.

\section{Indagación}

\subsection{Factibilidad Operativa}
Evalúa si el personal y los usuarios finales podrán y querrán utilizar el sistema propuesto \cite{kendall2011}. Es crucial para evitar la resistencia al cambio organizacional.

\subsection{Factibilidad Técnica}
Determina si la organización posee o puede adquirir el hardware, software y conocimiento técnico necesario para la ejecución del proyecto \cite{pressman2010}.

\subsection{Factibilidad Económica}
Consiste en la comparación de los costos del sistema con los beneficios proyectados \cite{sapag2011}. Se utilizan indicadores como el Valor Actual Neto (VAN):
$$VAN = \sum_{t=1}^{n} \frac{CF_t}{(1+r)^t} - I_0$$

\subsection{Factibilidad Comercial}
Analiza el mercado meta para confirmar que existe una demanda insatisfecha o una oportunidad de negocio real para la solución propuesta \cite{baca2013}.

\subsection{Factibilidad Legal}
Asegura que el proyecto no infrinja normativas de propiedad intelectual, leyes laborales o de protección de datos personales \cite{sapag2011}.

\section{Conclusión}
La integración de estos estudios garantiza que el proyecto no solo sea una buena idea técnica, sino una solución sostenible y rentable para la organización.
